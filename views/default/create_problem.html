Create a Problem

Instructions

To create a problem, fill out the form below:
  • 1.  Type the problem name (be as specific and concise as possible)
    • ex: Density (given mass and volume)
  • 2.  Type a comma-separated list of variables that the worksheet will randomly generate and change for each problem
    • ex: mass, volume
  • 3.  Type the prompt that will display each time the problem is shown; surround each variable name with curly braces {} to indicate a variable
    • ex: What is the density of an object with a mass of {mass} g and a volume of {volume} L?
  • 4.  Type the solution algorithm using the variable names in curly braces, any of the supported operations, and any integers/decimals needed
    • ex: {mass} / {volume}
  • 5.  Once the problem is created, you can go back to the home page to assign it to classes and edit variable ranges.

  • * Decimals between 0 and 1 must have a 0 before the decimal point! ex. 0.5
  • * Make sure the number of variables matches the number in the prompt and solution algorithm!
